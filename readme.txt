== Description ==

We can use this plugin for Tracking Inbound and outbound traffic! by page or post.
Through this we can enable any page or post for track.its give you a panel for track all pages trafic.



== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload track-site-traffic` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.set its setting admin->setting->trackit
4.Edit any post/page and Enable traffic check box.
